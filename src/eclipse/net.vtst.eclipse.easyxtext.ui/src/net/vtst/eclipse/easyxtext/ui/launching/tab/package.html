Configuration tabs for launching configurations defined with {@link net.vtst.eclipse.easyxtext.ui.launching.EasyLaunchConfigurationHelper}.

This package provides the class {@link net.vtst.eclipse.easyxtext.ui.launching.tab.EasyLaunchConfigurationTab}, which allows to easily define a configuration tab for a launch configuration defined with {@link net.vtst.eclipse.easyxtext.ui.launching.EasyLaunchConfigurationHelper}.

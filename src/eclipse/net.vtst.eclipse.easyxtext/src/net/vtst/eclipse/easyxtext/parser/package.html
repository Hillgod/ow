Extensions for parsers generated by Xtext/ANTLR.

This package provides two independent features:

  • A framework to customize the lexers generated by Xtext/ANTLR using Java regular expression. This framework is particularly useful for languages which require context-dependent lexing. See the class {@link net.vtst.eclipse.easyxtext.parser.EasyScanner}.
  • A value converter which checks the absence of whitespaces.
